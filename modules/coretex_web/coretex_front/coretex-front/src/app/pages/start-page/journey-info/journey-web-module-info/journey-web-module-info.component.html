After creating core module we need to create web module that will contain controller logic and front-end related logic.
For this purposes CoreTex has an ability to generate web module that use Angular 11.
Next command will create web module that will be deployed automatically once system start.

./gradlew moduleGen -Darchetype=web_angular_11_module -DprojectPath=/modules/todolist -DprojectName=todolistfront -DmoduleName=todoListFront -DangularProjectName=todoListFront -DpackagePath=com/example/project -DwebContextPath= --no-daemon

Created module contains completely configured sample angular web project
